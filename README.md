# GQ GENO — Official Website

Single-page site for the rap artist **GQ GENO**, with embedded Apple Music
players and an animated cosmic / hip-hop themed background (spinning
vinyl-record sun, drifting nebulae, floating boombox/cassette/mic, and a live
audio equalizer — all pure CSS/SVG, no image assets).

Live: https://eugenewilliams2.github.io/gq-geno/

## Develop
Just open `index.html` in a browser, or serve the folder:
```
python3 -m http.server 8000
```
